## Installation et Lancement

Pour faire fonctionner le jeu **Overgrowth**, suivez les étapes ci-dessous dans votre terminal :

### 1. Installation des dépendances
Le projet nécessite les bibliothèques `Pygame` et `NumPy`. Vous pouvez les installer automatiquement grâce au fichier des dépendances : avec les commandes *pip install pygame==2.1.3* et *pip install numpy*



```bash
pip install -r requirements.txt
```

finalement lancer le jeux avec la commande *python overgrowth.py*
